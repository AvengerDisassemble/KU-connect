# Backend 
## How to run
1. Install the packages using this command:
```bash
npm i
```
2. Generate the prisma files:
```bash
npx prisma generate
```
3. Copy the .env.example file to a .env file.
4. Execute the command:
```bash
npm run start
```

## How to access
You can access the api via
```
http://localhost:3000/api/ + The path
```
